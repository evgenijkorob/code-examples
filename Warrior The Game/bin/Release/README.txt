Это финальная версия продукта. Отныне он больше не поддерживается.

Управление осуществляется стрелками и клавишей Enter.

Для нормального отображения кириллицы, необходимо в командной строке настроить шрифт Lucida Console: 
щелкаем правой кнопкой по заголовку программы, заходим в свойства, в закладке "Шрифт" выбираем Lucida Console и нажимаем "Ок".

ПРИМЕЧАНИЕ:
При загрузке "Арены" может возникать ошибка памяти. Это никак не лечится - просто перезапустите программу.
